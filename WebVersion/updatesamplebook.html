How to Update the Accessible Image Sample Book (HTML version)

Connect to the Diagram Center’s SFTP

You will need a SFTP client. On the Mac, I use Cyberduck, which is pretty accessible with VoiceOver and keyboard access. I hear good things about the accessibility of FileZilla for Windows and Linux systems.

You will need to be given a username and password for sftp://diagramcenter.wpengine.com - you can see Diagram Center staff and Benetech’s IT department for this. You just enter the username, server name, and password to connect - all other settings are likely to be the defaults.

Once signed into SFTP, download the samplebook folder

If you need to edit more than one file. I encourage you to download the whole thing and make your edits on your computer, then delete the original and reupload the whole thing. If you need to edit just one, it's okay to just get that file, or even to edit it on the live server using your FTP program's tools if it has one.

Adding new pages and editing the page number structure

You'll need a text editor to do this; you probably have a favorite already. I personally use Sublime Text 2 which is fairly usable with VoiceOver but not perfectly so, unfortunately. TextMate 2.0 is an excellent choice for accessibility on the Mac. Visual Studio Express or TextPad is pretty accessible with JAWS on Windows.

The following is the HTML structure for the page numbers: 


<p>

 <a href="<nameofnextpage>.html">Next Page</a>|<a href="<nameofpreviouspage>.html">Previous Page</a>|<a href="index.html">Table of Contents</a>|<a href="http://diagramcenter.org/">Diagram Center Home</a>

</p>

If you get a new page or an updated version of an existing page, you will need to manually add this code twice to the page, once right after the opening body tag and once right before the closing body tag. If the book gets new pages, don't forget that you'll need to update the names of the next and previous pages for the pages before and after the new page, as well as the new page and the index.

Test. No, really, test.

It is super easy to muck these links up. Make sure the ones you changed all work before you upload the book to the server.

Upload the whole book back to the FTP

Make sure to delete the old version first, if necessary. If your FTP client has a synchronize command, you can also use that. You're done!
