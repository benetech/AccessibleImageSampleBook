How to Update the Accessible Image Sample Book (HTML version)

Grab the content from GitHub

You will need to visit the GitHub repository to obtain a copy of the master documents for the Web Version. Be sure to check out the code and commit it properly when you are done.

Connect to the Diagram Center’s SFTP

You will need a SFTP client. On the Mac, I use Cyberduck, which is pretty accessible with VoiceOver and keyboard access. I hear good things about the accessibility of FileZilla for Windows and Linux systems.

You will need to be given the host names, username, and password for the FTP - you can see Diagram Center staff and Benetech’s IT department for this. You just enter the username, server name, and password to connect, all other settings are likely to be the defaults.

Adding new pages and editing the page number structure

It is always best to update both the Web Version and the EPUB at the same time. Be sure that all of your changes are made in both versions of the files, and that you update all of the version numbers (Title Page, OPF, and Copyright page).

You'll need a text editor to make changes to the files; you probably have a favorite already. I personally use Sublime Text 2 which is fairly usable with VoiceOver but not perfectly so, unfortunately. TextMate 2.0 is an excellent choice for accessibility on the Mac. Visual Studio Express or TextPad is pretty accessible with JAWS on Windows.

The following is the HTML for the page navigation:


<nav><p><a href="nameofnextpage.xhtml">Next Page</a>&#xa0;&#xa0;|&#xa0;&#xa0;<a href="nameofpreviouspage.xhtml">Previous Page</a>&#xa0;&#xa0;|&#xa0;&#xa0;<a href="nav.xhtml">Table of Contents</a>&#xa0;&#xa0;|&#xa0;&#xa0;<a href="http://diagramcenter.org/">Diagram Center Home</a></p></nav>

In some cases, you may just need to replace all or most of the current Web Version with a new copy based on the latest EPUB file's contents. Be sure to add the nav bar to the top of each page

Test. No, really, test.

It is super easy to muck these links up. Make sure the ones you changed all work before you upload the book to the server.

Upload the whole book back to the FTP

Make sure to delete the old version first, if necessary. If your FTP client has a synchronize command, you can also use that. You're done!
